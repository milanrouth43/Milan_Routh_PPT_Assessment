# Task 20: Azure VNet (Basic Networking Practical)

## Objective
To understand basic cloud networking in Azure by configuring a Virtual Network and Subnets.

## Steps Taken
1. Created a Virtual Network named Milan-VNet in the RG-Capg resource group.
2. Configured the IP Address space as 10.0.0.0/16.
3. Created a dedicated subnet named Public-Subnet (10.0.1.0/24).
