# Task 21: Entra ID (Users, Roles, Basic Identity Management)

## Objective
To understand identity and access management (IAM) by creating users and assigning roles in Microsoft Entra ID.

## Steps Taken

1. Created a new user named "MilanUser" in Microsoft Entra ID.
2. Navigated to the Resource Group (RG-Capg) to manage access control.
3. Assigned the "Reader" role to the new user, granting read-only access to the resource group.

## esources Managed

- Service: Microsoft Entra ID
- User Created: MilanUser
- Role Assigned: Reader
