# Task 19: Azure Storage Account (Blob, File, Table)

## Objective
To learn multi-storage services in Azure including Blob, File, and Table storage.

## Steps Taken
1. Created a Storage Account named milanstorage123 in the RG-Capg resource group.
2. Created a Blob Container named images to store objects.
3. Configured an Azure File Share named myfiles.
4. Created a Table named employeedata for structured storage.

## Tools Used
- Azure Portal
- Azure Storage Account
